# 📓 Mi Blog Personal 

¡Bienvenido a mi espacio de reflexión! Este es un blog minimalista y moderno, diseñado para capturar pensamientos, ideas y aprendizajes diarios con una interfaz limpia y profesional.

---

## ✨ Características Principales
* **Diseño Premium**: Interfaz moderna, utilizando CSS Grid y Flexbox para un control total del diseño.
* **Gestión de Pensamientos**: CRUD completo (Crear, Leer, Editar y Eliminar) para organizar tus notas personales.
* **Glassmorphism & Gradients**: Estética visual atractiva con degradados vibrantes y tarjetas con sombras suaves.
* **Totalmente Responsivo**: Adaptado para visualizarse perfectamente en computadoras de escritorio.

---

## 🚀 Tecnologías Utilizadas
* **Backend**: Python con el micro-framework **Flask**.
* **Frontend**: HTML5 puro y CSS3 personalizado (sin librerías externas).
* **Base de Datos**: PostgreSQL para un almacenamiento ligero y eficiente de los posts con respaldo en la nube utilizando Supabase.
* **Favicon Dinámico**: SVG embebido para una identidad visual nítida.
